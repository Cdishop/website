Generating Time in a Data Frame

There are two code variations I use to generate time indexes. If I need time cycles

##   id    score time
## 1  a 19.02649    1
## 2  b 22.25877    2
## 3  c 20.93711    3
## 4  a 29.49510    1
## 5  b 15.34409    2
## 6  c 19.38187    3
## 7  a 15.42013    1
## 8  b 21.72485    2
## 9  c 20.10647    3

then I use a sequence command.

time <- seq(from = 1, to = 3, each = 3)

If I need time ordered

##   id    score time
## 1  a 21.44159    1
## 2  a 17.64815    1
## 3  a 22.00805    1
## 4  b 23.49984    2
## 5  b 18.92340    2
## 6  b 19.23177    2
## 7  c 14.93441    3
## 8  c 19.72774    3
## 9  c 16.69150    3

then I use a replicate command.

time <- rep(c(1:3), each = 3)
